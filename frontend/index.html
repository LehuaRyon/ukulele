Ukulele Song Lineup


Learn Songs on the Ukulele!

Keep track of all the songs you would like to learn on the ukulele in one place! Each song will be categorized by the genre it belongs to, which you can change at any time.

The purpose of this application is to keep a record of songs that you would like to learn in the future, if you can't spare the time to do so right now. Each song can also be referred back to for chords when needed so you don't have to rely purely on memory! You can add as many songs as you would like and edit any song's information at any time. Feel free to add and/or delete as many songs on your list.

Happy strumming!
